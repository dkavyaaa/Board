# Board AI Dashboard

Board is a full-stack web application that uses OpenAI's GPT API to generate actionable tasks from meeting transcripts. It features a React frontend with Material UI and a Node.js + Express backend with Prisma ORM connected to a PostgreSQL database.

---

## Features

- Paste meeting transcripts and generate action items automatically using OpenAI GPT.
- View, update (mark complete), and delete action items.
- Visualize progress with a pie chart showing completed vs pending tasks.
- Backend API built with Express and Prisma ORM.
- PostgreSQL database for persistent storage.
- Modular, maintainable codebase with TypeScript.

---

## Tech Stack

- **Frontend:** React, TypeScript, Material UI, Recharts, Axios  
- **Backend:** Node.js, Express, TypeScript, Prisma ORM, PostgreSQL  
- **AI:** OpenAI GPT-3 API (text-davinci-003)  
- **Database:** PostgreSQL  

---

## Getting Started

### Prerequisites

- Node.js v16 or later  
- npm (comes with Node.js)  
- PostgreSQL database (local or hosted)  
- OpenAI API key ([Get API key](https://platform.openai.com/account/api-keys))
